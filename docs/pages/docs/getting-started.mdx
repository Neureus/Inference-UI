## What is Inference UI?

Inference UI is an AI-native cross-platform UI component library that integrates intelligence into every component. Unlike traditional UI libraries where AI is an afterthought, Inference UI is built from the ground up with AI capabilities baked into each component.

## Prerequisites

Before you begin, ensure you have:

- **Node.js 18+** installed
- **React Native 0.70+** or **React 18+**
- **TypeScript 5+** (recommended but not required)
- Basic knowledge of React or React Native

<Callout type="warning">
  Inference UI currently supports **React Native** with React (web) support coming in Phase 3.
</Callout>

### Setup providers

Wrap your app with Inference UI providers:

```tsx filename="App.tsx"
import React from 'react';
import { EventProvider } from '@inference-ui/events';
import { FlowProvider } from '@inference-ui/flows';

export default function App() {
  return (
    <EventProvider
      config={{
        batchSize: 50,
        batchInterval: 20000,
        endpoint: 'https://your-worker.workers.dev/events',
      }}
    >
      <FlowProvider>
        <YourApp />
      </FlowProvider>
    </EventProvider>
  );
}
```

## Your First Inference UI Component

Here's a complete example of a simple form with AI-powered validation:

```tsx filename="SmartForm.tsx"
import React, { useState } from 'react';
import { View } from 'react-native';
import {
  GradientBackground,
  GlassCard,
  GlassText,
  AIInput,
  AIButton,
} from '@inference-ui/react-native';
import { useEventTracker, useFormTracking } from '@inference-ui/events';

export function SmartForm() {
  const [name, setName] = useState('');
  const [email, setEmail] = useState('');
  const track = useEventTracker();
  const { trackSubmit } = useFormTracking('smart_form');

  const handleSubmit = async () => {
    // Your submit logic
    await submitForm({ name, email });

    // Track successful submission
    trackSubmit(true);
    track('form_success', { formType: 'smart_form' });
  };

  return (
    <GradientBackground gradient="aurora">
      <GlassCard glassStyle="medium" padding={6}>
        <GlassText variant="heading" size="2xl" weight="bold">
          Get Started
        </GlassText>

        <AIInput
          name="name"
          label="Full Name"
          value={name}
          onChange={setName}
          enableValidation={true}
          placeholder="John Doe"
        />

        <AIInput
          name="email"
          label="Email Address"
          value={email}
          onChange={setEmail}
          enableValidation={true}
          enableAutocomplete={true}
          placeholder="john@example.com"
          keyboardType="email-address"
        />

        <AIButton
          title="Submit"
          onPress={handleSubmit}
          variant="primary"
          size="lg"
          fullWidth
        />
      </GlassCard>
    </GradientBackground>
  );
}
```

### What's Happening Here?

1. **GradientBackground**: Provides beautiful animated gradient background
2. **GlassCard**: Creates glassmorphism effect container
3. **AIInput**:
   - Validates input in real-time using local AI
   - Provides autocomplete suggestions
   - Automatically tracks focus/blur events
4. **AIButton**:
   - Tracks button presses automatically
   - Shows loading state during async operations
5. **Event Tracking**:
   - Form interactions tracked automatically
   - Custom events tracked with `useEventTracker`
